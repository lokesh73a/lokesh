Unit Converter CLI Tool
A simple, interactive command-line Python program for performing common unit conversions across length, weight, and time measurements. This project is ideal for learning fundamental Python concepts like functions, loops, user input, and conditional logic (if/elif/else).
Features
Length Conversion: Meters, Kilometers, Centimeters
Weight Conversion: Grams, Kilograms, Pounds
Time Conversion: Seconds, Minutes, Hours
Interactive Menu: Easy-to-use menu system to guide user interaction
Modular Code: Uses dedicated functions for each conversion type
Getting Started
These instructions will get you a copy of the project up and running on your local machine.
Prerequisites
You need Python installed on your system. You can download the latest version from the official Python website. This code runs on both Python 2 (with minor input() adjustments) and Python 3.
